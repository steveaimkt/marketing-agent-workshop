# 2교시 실습 · 내 트랙 대시보드 만들기

> 오늘 첫 결과물. 내 데이터를 넣으면 내 직군 성적표가 웹 화면으로 뜬다.
> 어려운 말은 그때그때 쉬운 말로 풀어 준다. 한 단계 끝나면 물어보고 다음으로 간다.
> 트랙은 다시 묻지 않는다. 1교시에서 고른 것을 그대로 쓴다.

## 진행 흐름 (3블록)

```
블록 1  구축   · 트랙 확인 → 파일 확인 → 화면 설계 확정        ⏸ 답 대기
블록 2  실행   · 계산 규칙 → 한 줄 명령 → 웹 화면 완성          ⏸ 답 대기
블록 3  다듬기 · 3~5번 고쳐서 회의에 낼 수 있게                 ⏸ 선택
```

---

## 블록 1 · 구축

### STEP 1-0. 트랙 확인 (다시 묻지 않는다)

```
1교시에서 고른 트랙 그대로 갑니다.
지금 만드는 것은 "{트랙명} 대시보드"입니다.

  · 화면 구성이 트랙마다 다릅니다. 만드는 방법은 전원 같습니다.
  · 쓰는 데이터는 내-마케팅분석 폴더에 이미 들어 있습니다.
```
⏸ 맞으면 "맞아요", 트랙을 바꾸고 싶으면 바꿀 번호를 알려 주세요.

### STEP 1-1. 무엇을 만드나

```
📊 내 트랙 대시보드를 만듭니다.

🎯 무엇을 만드나
  "관리자 화면에 들어가 숫자를 긁고 엑셀에 붙이는 일"을 대신하는 작은 도구.
  내 데이터 파일을 넣으면 → 지표를 계산해 →
  한눈에 보는 성적표 웹 화면(대시보드)으로 보여 준다.

⏱ 이렇게 바뀐다
  · 전 : 관리자 접속 → 기간 설정 → 표 정리 → 엑셀 = 약 40분
  · 후 : 파일 넣고 한 줄 명령 = 1분, 결과는 예쁜 웹 화면

🧰 준비물 (1교시 산출물이면 충분)
  · 내-마케팅분석 폴더의 내 트랙 데이터 + CLAUDE.md + 1교시에서 만든 컬럼 분석표

💡 오늘 쓰는 쉬운 말
  · 지표 = 성적표 숫자 (전환율, 수익률 등)
  · 대시보드 = 성적표를 한눈에 보는 웹 화면
  · 스킬 = "이렇게 계산하고 이렇게 보여줘"를 적어 둔 내 업무 저장 파일
  · 이 도구는 계정을 건드리지 않는다. '보기·계산·표시'만 한다.
```
⏸ 여기까지 이해되셨으면 "다음"이라고 답해 주세요.

### STEP 1-2. 내 트랙 화면 구성 확정

클로드는 **고른 트랙의 표 하나만** 출력한다.

**트랙 1 퍼포먼스** (위에서부터 순서대로 쌓는다)
1 KPI 카드 · 광고비 · 노출 · 클릭 · 전환 · ROAS · CPA · CTR ← campaigns.csv
2 캠페인 성과표 · 캠페인별 지표 정렬 + 기준 위반 빨강 ← campaigns.csv
3 추세 차트 · 기간별 광고비 대비 매출 흐름 ← daily.csv
4 매체 비교 · 메타 · 구글 · 네이버 효율 비교 ← daily.csv
5 소재 TOP·BOTTOM · 상위 5개, 하위 5개 소재 ← creatives.csv

**트랙 2 콘텐츠**
1 콘텐츠 성과표 · 콘텐츠별 조회 · 체류 · 전환 ← contents.csv
2 키워드 트렌드 차트 · 검색량 추이와 급상승 키워드 ← keywords.csv
3 채널 비교 · 채널별 콘텐츠 효율 ← contents.csv
4 주간 캘린더 · 발행 예정 콘텐츠 일정 ← contents.csv
5 카피 후보 카드 · 제목 · 카피 변형안 비교 ← contents.csv

**트랙 3 CRM**
1 고객 요약 카드 · 전체 고객 수 · 신규 · 재구매율 · 평균 구매주기 ← customers.csv
2 세그먼트 분포 · 등급 · 구매 빈도 · 금액대별 분포 ← customers.csv
3 LTV 분석 · 세그먼트별 생애가치와 기여 매출 ← orders.csv
4 이탈 위험 리스트 · 최근 구매 이후 경과일 기준 위험군 ← orders.csv
5 재구매 패턴 · 첫 구매 이후 재구매까지의 흐름 ← orders.csv

**트랙 4 브랜드**
1 경쟁사 변경점 · 사이트 · SNS · 메시지 변경 감지 결과 ← competitor-snapshots/
2 트렌드 카드 · 브랜드 관련 키워드 급상승 ← reviews.csv
3 VoC 분류 · 리뷰 · 댓글 감정 및 주제 분류 ← reviews.csv
4 페인포인트 TOP 5 · 가장 많이 언급된 불만 · 요청 ← reviews.csv
5 브랜드 보이스 체크 · 우리 메시지의 톤 일관성 점수 ← brand-guide.md

**트랙 5 그로스**
1 트래픽 요약 · 세션 · 사용자 · 신규 비율 ← sessions.csv
2 채널 기여 · 유입 채널별 전환 기여 ← sessions.csv
3 퍼널 도식 · 방문 → 조회 → 장바구니 → 구매 단계별 이탈률 ← funnel.csv
4 이탈 급증 알림 · 전주 대비 이탈이 커진 단계 강조 ← funnel.csv
5 랜딩 성과표 · 페이지별 유입 대비 전환 ← landing.csv

**트랙 6 1인마케터**
1 주간 요약 카드 · 광고비 · 매출 · 전환 · 콘텐츠 발행 수 ← weekly.csv
2 광고 미니 리포트 · 상위 · 하위 캠페인 각 3개 ← weekly.csv
3 콘텐츠 미니 리포트 · 이번 주 성과 상위 콘텐츠 ← weekly.csv
4 할 일 제안 · 이번 주 우선순위 3개 ← weekly.csv · tasks.md

포인트: "화면이 5개라고 겁먹지 마세요. 위에서부터 2개만 나와도 성공입니다. 나머지는 블록 3에서 붙입니다."
⏸ 내 화면 구성이 확인되면 "블록 2"라고 답해 주세요.

## 블록 2 · 실행

### STEP 2-1. 계산 규칙 정하기

따라하기:
```
내-마케팅분석 폴더의 데이터로, 내 트랙 대시보드에 들어갈 지표를 계산해줘.
계산식과 어떤 칸을 썼는지도 같이 보여줘
```
클로드 실행 규칙: 트랙별 핵심 지표를 계산식과 함께 표로 먼저 제시하고 확인을 받는다. CLAUDE.md의 '우리만의 기준'이 있으면 그 기준으로 좋고 나쁨(신호등)을 판정한다. 기준이 없으면 기본값을 제안하고 동의를 구한다.

신호등 기본값(트랙별로 클로드가 알아서 맞춘다):
- 퍼포먼스: ROAS 2.0 미만 노랑, 1.0 미만 빨강
- 콘텐츠: 전환율 1% 미만 노랑, 0.5% 미만 빨강
- CRM: 마지막 구매 60일 초과 노랑, 90일 초과 빨강
- 브랜드: 부정 언급 10건 이상 노랑, 20건 이상 빨강
- 그로스: 단계 이탈률 25% 이상 노랑, 40% 이상 빨강
- 1인마케터: 전주 대비 매출 감소 노랑, 2주 연속 감소 빨강

한 줄 붙이기: 목적이 브랜딩·도달인 항목은 수익률만으로 판단하지 않는다.
⏸ 계산 결과가 납득되면 "다음"이라고 답해 주세요.

### STEP 2-2. 화면으로 만들기

따라하기:
```
방금 계산 결과를 대시보드 웹 화면(HTML 한 장)으로 만들어줘.
맨 위에 큰 숫자 카드, 가운데 성과표와 차트, 맨 아래 다음에 할 일 목록.
기준 위반 항목은 빨강으로 표시하고, 만들고 브라우저로 열어줘
```

이런 화면이 나오면 성공:
```
┌──────────────────────────────────────────┐
│  [카드]  [카드]  [카드]  [카드]            │  ← 큰 숫자 요약
├──────────────────────────────────────────┤
│  성과표 (빨강으로 표시된 줄이 섞여 있음)      │
├──────────────────────────────────────────┤
│  차트 또는 도식                             │
├──────────────────────────────────────────┤
│  다음에 할 일 3가지                         │
└──────────────────────────────────────────┘
```
포인트: "색이나 순서가 조금 달라도 이 형태면 성공입니다. 완벽한 화면은 블록 3에서 만듭니다."
⏸ 웹 화면이 떴으면 "됐어"라고 답해 주세요. 안 되면 어떤 화면인지 알려 주세요.

## 블록 3 · 다듬기

한 번에 다 고치지 않는다. 한 줄씩 3~5번 고친다.

자주 쓰는 고치기 문장:
```
기준선을 그어줘. 우리 기준 수치가 어디인지 보이게
빨강으로 표시된 항목만 위로 올려줘
숫자에 천 단위 콤마 넣고 소수점은 한 자리로 줄여줘
맨 아래 '다음에 할 일'을 3개로 늘리고 이유도 한 줄씩 붙여줘
모바일에서도 안 깨지게 만들어줘
```

트랙별로 한 가지씩 더:

| 트랙 | 추천 다듬기 |
|:--|:--|
| 1 퍼포먼스 | 소재 TOP·BOTTOM 카드를 아래에 추가해줘 |
| 2 콘텐츠 | 주간 캘린더를 표로 아래에 추가해줘 |
| 3 CRM | 이탈 위험 고객 리스트를 아래에 추가하고 고객 ID만 표시해줘 |
| 4 브랜드 | 페인포인트 TOP 5 카드를 위쪽으로 올려줘 |
| 5 그로스 | 퍼널 도식에 전주 대비 변화를 화살표로 표시해줘 |
| 6 1인마케터 | 이번 주 우선순위 3개를 맨 위 카드 옆에 붙여줘 |

⏸ 마음에 드는 화면이 나왔으면 "완성"이라고 답해 주세요.

## 완료

```
✅ 방금 만든 것 = 내 데이터 → 내 직군 성적표 웹 화면.

🔁 내 업무에 쓰는 법
  매주 데이터 파일만 바꿔 넣으면 같은 성적표가 1분 만에 나온다.

📌 지금 이건 아직 '내가 파일을 줘야' 도는 정적 도구다.
  · 3교시 : 내 계정을 연결해서, 파일 없이도 실제 데이터를 불러오게 만든다.
  · 4교시 : 그걸 '명령 한 줄이면 알아서 도는 에이전트'로 키운다.
```

산출물: `내-마케팅분석/대시보드.html` + 계산 규칙과 신호등 기준(4교시 에이전트에 그대로 들어감)
다음은 3교시 "내 계정 연결하기(MCP 설치)" 입니다.

## 트러블슈팅

| 증상 | 해결 |
|:--|:--|
| 데이터 파일을 못 찾음 | "내-마케팅분석 폴더에 뭐가 있는지 목록 보여줘"로 먼저 확인 |
| 칸 이름이 달라서 계산 실패 | "이 칸이 광고비야"처럼 말로 알려 주면 된다 |
| 화면이 텅 비어 보임 | "데이터가 실제로 들어갔는지 표로 먼저 보여줘"로 되돌려 확인 |
| HTML이 안 열림 | 탐색기에서 html 우클릭 → 브라우저로 열기 |
| 화면 5개가 다 안 나옴 | 2개만 나와도 성공. 나머지는 블록 3에서 한 줄씩 추가 |
| 트랙을 바꾸고 싶음 | 1교시 블록 2로 돌아가 데이터부터 다시 복사 |
