# 3교시 실습 · 내 계정 연결하기 (MCP 설치)

> MCP = 내 계정과 프로그램을 잇는 '연결선'. 이게 있으면 파일을 직접 안 줘도
> 클로드코드가 계정에서 데이터를 알아서 가져온다. 한 단계씩 확인하며 진행한다.

## 🎯 이 시간의 핵심 (먼저 읽는다)

```
연결은 두 층이다. 결과를 담을 곳(노션·디스코드 웹훅)은 전원 붙이고,
데이터를 읽을 곳(메타·네이버 검색광고)은 계정이 있는 사람만 붙인다.

· 1층 노션과 디스코드 웹훅은 전원 붙인다. 로그인 동의와 주소 복사라 10분이면 끝난다.
· 2층 데이터 연결은 메타 광고와 네이버 검색광고 2종뿐이다. 내 직군에 맞는 것을 제안해 준다.

· 구글 계열(구글애즈·GA4·구글시트)은 오늘 하지 않는다.
  클라우드 프로젝트 생성, API 사용 설정, 서비스 계정 키 발급, 속성 권한 부여까지
  단계가 길어서 한 시간을 다 써도 못 끝내는 사람이 나온다.
  가이드를 드리니 회사에 돌아가서 같은 순서로 붙이면 된다.

· 2층을 아예 안 해도 된다. 오늘 실습은 전부 실습 파일 기준으로 진행한다.
  연결은 '해보면 좋은 것'이지 '해야만 하는 것'이 아니다.
  4교시 에이전트도, 5교시 액션 제안도 실습 파일로 똑같이 완성된다.

· 10분 안에 안 풀리면 손을 든다. 그 자리에서 접고 진도를 나간다.
  못 붙였다고 손해 보는 게 없다.
```

## 진행 흐름 (3단계)

```
1단계  소개  · 연결이 왜 필요한가 + 두 층 안내 + 내 직군 제안   ⏸ 답 대기
2단계  실행  · 1층(전원) → 2층(선택) 설치하고 작동 확인      ⏸ 답 대기
3단계  활용  · 안전 규칙 + 못 한 연결 자습 안내              ⏸ 선택
```

## 1단계 · 소개

```
🔌 내 계정을 연결합니다 (MCP 설치).

🎯 왜 필요한가
  2교시 대시보드는 '내가 파일을 넣어야' 돈다. 매주 파일 내려받는 것도 일이다.
  계정을 연결하면, 파일 없이 클로드코드가 계정에서 최신 데이터를 직접 가져온다.

🙅 연결이 필요 없는 경우도 많다
  · 어차피 매달 한 번만 보는 데이터라면, 파일(CSV)로 충분하다.
  · 회사 보안상 API 키 발급이 막혀 있다면, 파일로 간다. 결과물은 똑같이 나온다.
  · 내 데이터를 아직 안 가져오셨다면, 실습 파일로 간다. 배우는 건 똑같다.

💡 쉬운 말
  '연결(MCP)'은 계정에 '보기 권한'을 주는 것이다.
  열쇠를 새로 만드는 게 아니라, 보기용 출입증을 발급받아 붙이는 느낌이다.
```

**오늘 붙이는 것은 두 층이다**

```
[1층] 결과를 담을 곳     노션 + 디스코드 웹훅     전원 공통 · 약 10분
[2층] 데이터를 읽을 곳   메타 또는 네이버 검색광고   직군별 제안 · 선택 · 약 10~15분
```

1층은 4교시 에이전트가 리포트를 저장하고 요약을 보낼 곳이라 **전원 같이 붙인다.** 로그인 동의와 주소 복사라 빠르다.
2층은 내 계정에서 데이터를 읽는 것이라 **계정이 있는 사람만** 한다.

**직군별 제안 (2층, 택1 또는 생략)**

| 내 트랙 | 현장에서 붙일 것 | 왜 이걸 권하나 | 내 트랙 본 데이터는 |
|:--|:--|:--|:--|
| 1 퍼포먼스 | **메타 광고** 또는 **네이버 검색광고** | 트랙과 그대로 직결된다. 붙이면 실계정 데이터로 이어서 만들 수 있다 | 붙인 매체가 곧 내 데이터 |
| 2 콘텐츠 | **네이버 검색광고** | 사람들이 실제로 검색하는 키워드와 그 단가를 볼 수 있다. 콘텐츠 주제 잡을 때 근거가 된다 | 데이터랩은 자습, 오늘은 실습 파일 |
| 3 CRM | **메타 광고** | 어떤 광고로 들어온 고객인지 연결해 보는 감각을 얻는다 | 구글 시트는 자습, 오늘은 실습 파일 |
| 4 브랜드 | **메타 광고** | 경쟁사 광고 소재를 계정 화면에서 직접 보는 경험이 된다 | 웹 수집은 자습, 오늘은 실습 파일 |
| 5 그로스 | **메타 광고** | 유입이 어디서 오는지 한쪽 끝을 실계정으로 본다 | GA4는 자습, 오늘은 실습 파일 |
| 6 1인 마케터 | **메타** 또는 **네이버 검색광고** 중 실제 쓰는 것 | 혼자 운영하는 매체가 곧 내 데이터다 | 붙인 매체가 곧 내 데이터 |

```
⚠️ 2층은 '내가 관리자 권한을 가진 계정'이 있어야 합니다.
   회사 계정인데 권한이 없으시면 건너뛰세요. 그게 정상입니다.

📌 2층을 건너뛰어도 오늘 결과물은 똑같이 나옵니다.
   트랙 데이터는 어차피 실습 파일로 돌아갑니다.
   2층은 '연결이란 이런 것이구나'를 손으로 한 번 해보는 시간입니다.
```

⏸ 1층부터 갑니다. "노션 연결하자" 라고 답해 주세요.

## 2단계 · 실행

어떤 연결이든 순서는 똑같다. **무엇을·왜 → 키 발급 → 설치 → 작동 확인.**
이 4단계를 지키면 어떤 도구를 만나도 같은 방식으로 붙일 수 있다.

# 1층 · 결과를 담을 곳 (전원 공통, 약 10분)

### 1-1. 노션 연결 (기본)

- **무엇을 왜** : 4교시 에이전트가 만든 리포트를 노션에 자동으로 쌓는다. 매주 결과가 한곳에 모인다.
- **키 발급** : 없다. 로그인하고 어느 페이지를 쓸지 고르면 끝난다.
- **설치** :
  ```
  노션 연결해줘
  ```
  브라우저가 열리면 로그인하고, 에이전트가 쓸 페이지를 선택한다.
  ⚠️ 회사 노션이 아니라 **개인 노션**으로 하셔도 됩니다. 오늘은 연습이니까요.
- **작동 확인** :
  ```
  내 노션에 "연결 테스트" 라는 페이지 하나 만들어줘
  ```
- ✅ 성공 : 노션에 페이지가 생기면 된 것이다.

✅ 체크: "노션에 페이지가 생겼나요?"

### 1-2. 디스코드 웹훅 연결 (기본)

디스코드는 앱을 붙이지 않고 **웹훅 주소 하나만** 쓴다. 주소를 복사해 붙여넣으면 끝이라 제일 빠르다.

- **무엇을 왜** : 리포트 요약을 휴대폰으로 받는다. 노트북을 안 봐도 결과를 확인할 수 있다.
- **주소 발급** : 디스코드에서 내 서버를 하나 만들고(없으면), 채널 설정에서 웹훅 주소를 복사한다.
  ```
  디스코드 → 서버 만들기 (없으면) → 채널 우클릭 → 채널 편집
  → 연동 → 웹후크 → 새 웹후크 → 웹후크 URL 복사
  ```
- **설치** : 복사한 주소를 붙여넣으며 이렇게 말한다.
  ```
  디스코드 웹훅 연결해줘. 주소는 이거야
  ```
- **작동 확인** :
  ```
  디스코드로 "연결 테스트" 한 줄만 보내줘
  ```
- ✅ 성공 : 디스코드 채널에 메시지가 뜨면 된 것이다. 휴대폰에도 알림이 온다.

```
💡 웹훅이 뭔가요
   '이 주소로 메시지를 보내면 이 채널에 뜬다'는 전용 주소입니다.
   로그인도, 봇 설치도 필요 없습니다. 주소 하나면 끝입니다.

⚠️ 웹훅 주소는 비밀번호처럼 다룹니다. 주소를 아는 사람은 누구나 그 채널에 글을 쓸 수 있습니다.
   화면 공유 중에는 주소를 띄우지 마세요.
```

✅ 체크: "디스코드에 메시지가 도착했나요?"

⏸ 1층이 끝났으면 "1층 완료" 라고 답해 주세요. 2층은 선택입니다.

# 2층 · 데이터를 읽을 곳 (직군별 제안, 선택)

### A. 메타 광고 연결 (약 10분)

- **무엇을 왜** : 페이스북·인스타 광고 성과(광고비·노출·클릭·전환)를 계정에서 바로 읽는다.
- **키 발급** : 별도 키 입력이 없다. 브라우저에서 로그인하고 동의만 누르면 된다.
- **설치** : 클로드코드 채팅창에 이렇게 말한다.
  ```
  메타 광고 계정 연결해줘
  ```
  브라우저가 열리면 직접 로그인하고 권한 동의를 누른다.
- **작동 확인** :
  ```
  내 메타 광고 계정 목록이랑 지난 7일 캠페인 성과 보여줘
  ```
- ✅ 성공 : 계정 이름과 캠페인 몇 개가 표로 나오면 된 것이다. 숫자가 적어도 상관없다.
- ⚠️ 막히는 지점 : 광고 계정 권한이 '분석자' 이상이어야 보인다. 권한이 없으면 2층을 건너뛴다.

### B. 네이버 검색광고 연결 (약 15분)

- **무엇을 왜** : 네이버 검색광고 성과와 키워드별 지표를 계정에서 바로 읽는다.
- **키 발급** : 네이버 검색광고 관리 화면에서 **키 3개를 복사**한다. 별도 심사나 승인 대기가 없다.
  ```
  네이버 검색광고 → 우측 상단 '도구' → 'API 사용 관리'
  → 액세스 라이선스(API_KEY) · 비밀키(SECRET_KEY) · CUSTOMER_ID 복사
  ```
- **설치** : 복사한 키를 채팅창에 붙여넣으며 이렇게 말한다.
  ```
  네이버 검색광고 연결해줘. 키는 이거야
  ```
  ⚠️ 키를 붙여넣을 때는 화면 공유를 잠깐 꺼 주세요.
- **작동 확인** :
  ```
  지난 7일 네이버 검색광고 캠페인 성과 보여줘
  ```
- ✅ 성공 : 캠페인명과 노출·클릭·비용이 나오면 된 것이다.
- ⚠️ 막히는 지점 : 검색광고 계정과 네이버 아이디가 다를 수 있다. 광고주 계정으로 로그인해야 API 메뉴가 보인다.

### C. 2층 건너뛰기 (0분)

```
✅ 2층은 건너뜁니다. 1층(노션·디스코드)만으로 충분합니다.

지금 하실 일은 하나입니다. 강사 시연을 보면서
'연결하면 이런 게 되는구나'만 눈으로 확인하세요.

그리고 이 질문 하나만 스스로 답해 보세요.
  "회사에 돌아가면, 내가 제일 먼저 붙이고 싶은 계정은 무엇인가?"

그 답을 메모해 두시면 3단계에서 그 연결 가이드를 안내해 드립니다.
실습은 지금까지처럼 데이터/ 폴더의 실습 파일로 그대로 갑니다.
```

### 연결 상태 점검

```
지금 연결된 것들 목록이랑 상태 보여줘
```

⏸ 2층까지 됐으면 "연결됨", 1층만 하고 넘어가려면 "1층만" 이라고 답해 주세요.

## 3단계 · 활용

```
🛡 안전 규칙 (꼭 지킨다)
  · 계정은 '조회'만 한다. 예산 변경, 광고 집행, 메시지 발송, 콘텐츠 발행,
    삭제, 결제는 자동화하지 않는다.
  · 조치는 사람이 직접 하거나, 내가 명확히 시켰을 때만 한다.
  · 발급받은 키는 남에게 공유하지 않는다. 화면 공유 중에는 키 화면을 띄우지 않는다.
  · 고객 이름, 연락처, 주소, 결제정보가 담긴 자료는 지우고 다룬다.
```

**오늘 안 한 연결은 이렇게 붙인다 (자습 가이드)**

| 도구 | 무엇을 읽나 | 발급 절차 | 예상 소요 |
|:--|:--|:--|:--|
| 구글애즈 Google Ads | 검색광고 키워드·캠페인 성과 | OAuth 동의 + 개발자 토큰 신청 (승인 대기 있음) | 하루 이상 걸릴 수 있음 |
| GA4 구글 애널리틱스 | 방문·유입 채널·전환·퍼널 | 클라우드 프로젝트 → API 사용 설정 → 서비스 계정 키 → 속성 권한 부여 | 30분에서 2시간 |
| 구글 시트 Sheets | 고객·주문 등 시트 데이터 | 구글 로그인 동의 + 읽을 시트 지정 | 15분 |
| 네이버 데이터랩 | 검색 수요·트렌드 | 네이버 개발자센터 앱 등록 후 Client ID·Secret | 15분 |
| 웹 수집 | 경쟁사 페이지·리뷰 | 무료 API 키 1개 발급 | 10분 |
