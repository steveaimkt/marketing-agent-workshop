# 실습 파일 · 마케터를 위한 클로드코드 에이전트 구축

> 멋쟁이사자처럼 AIERS · 오프라인 6시간
> 이 폴더 하나만 받으면 오늘 실습이 전부 진행됩니다. 아무것도 준비해 오지 않아도 완주할 수 있습니다.

---

## 어떻게 쓰나요

안티그래비티에서 이 폴더를 연 뒤, 채팅창에 이렇게 입력하세요.

```
시작하자
```

그러면 `00_시작하기` 스킬이 열리고, 6개 마케터 직군 중 본인 트랙을 고르는 것부터 안내합니다.
한 번 고른 트랙은 1교시부터 5교시까지 그대로 이어집니다.

---

## 폴더 구성

```
실습파일/
├── 00_시작하기/          내 트랙 고르기 (여기서 시작)
├── 1교시_기초교육/        세팅 · 지침서 · 시각화 손풀기
├── 2교시_대시보드구축/     내 데이터를 웹 화면으로
├── 3교시_MCP설치/         계정 연결 체험 (메타 · 네이버 검색광고)
├── 4교시_에이전트구축/     명령 한 줄로 도는 에이전트
├── 5교시_후속액션/         다음 행동 제안 + 내 것으로 옮기는 방법
└── 데이터/
    ├── 1_퍼포먼스/        캠페인 30건 · 일자별 30일 · 소재 20종
    ├── 2_콘텐츠/          콘텐츠 40건 · 키워드 100개 · 경쟁사 게시물 30건
    ├── 3_CRM/            고객 500명 · 주문 내역 · 문의 60건 (익명 처리됨)
    ├── 4_브랜드/          리뷰 200건 · 경쟁사 스냅샷 2회분 · 브랜드 가이드 · 광고 소재 30건
    ├── 5_그로스/          방문 30일 · 퍼널 · A/B 실험 3건 · 랜딩 15개
    └── 6_1인마케터/       광고·콘텐츠·매출 4주치 · 반복 업무 15개
```

## 같이 쓰는 웹 도구

**마케팅 자동화 매트릭스** · https://likelion-marketing-automation-matrix.netlify.app

내 반복 업무 중 무엇부터 자동화할지 진단하는 도구입니다. 업무를 넣으면 우선순위가 매겨지고, 그것을 스킬로 만들지 에이전트로 만들지 대시보드로 만들지까지 정해 줍니다. 마지막에 나오는 프롬프트를 복사해 채팅창에 붙여넣으면 바로 구축으로 이어집니다.

4교시에 같이 씁니다. 미리 해보고 오시면 그만큼 빨라집니다. 입력값은 각자 브라우저에만 저장되고 서버로 전송되지 않습니다.

## 결과물은 늘 세 곳에 도착합니다

```
① HTML       내 컴퓨터에 열리는 웹 화면. 보고 쓰는 본체
② 노션        같은 내용을 페이지로 아카이브. 쌓여서 히스토리가 됨
③ 디스코드    핵심 요약과 노션 링크를 발송. 휴대폰으로 받음
```

트랙이 달라도, 만드는 에이전트가 달라도 이 구조는 같습니다. 그래서 3교시에 노션과 디스코드를 전원 연결합니다.

---

## 6개 트랙

| 번호 | 트랙 | 오늘 만드는 것 |
|:--|:--|:--|
| 1 | 퍼포먼스 마케터 | 광고 성과 대시보드 → 광고 분석 에이전트 |
| 2 | 콘텐츠 마케터 | 콘텐츠 성과 대시보드 → 콘텐츠 캘린더 에이전트 |
| 3 | CRM 마케터 | 고객 세그먼트 대시보드 → 고객 분석 에이전트 |
| 4 | 브랜드 마케터 | 브랜드 모니터링 대시보드 → 브랜드 모니터링 에이전트 |
| 5 | 그로스 마케터 | 퍼널 대시보드 → 퍼널 진단 에이전트 |
| 6 | 1인 마케터 | 주간 통합 대시보드 → 주간 브리핑 에이전트 |

당일에는 **한 트랙만** 실습합니다. 나머지 5개는 노션 가이드와 이 폴더의 데이터로 자습할 수 있습니다.


## 알아 두실 것

**실습 데이터는 전부 가상입니다.**
결로아라는 가상 스킨케어 브랜드를 기준으로 만든 데이터입니다. 실제 기업 데이터가 아닙니다.

**실습은 전부 실습 데이터로 진행합니다.**
회사 데이터를 가져오지 않으셔도 됩니다. 5교시에 이걸 내 회사 데이터와 내 계정으로 옮기는 방법을 순서대로 안내합니다. 실제 전환은 자리에 돌아가서 하시면 됩니다.

**개인정보는 반드시 지우고 쓰세요.**
본인 데이터를 쓰실 때 이름, 연락처, 주소, 결제정보는 제거한 뒤 사용하세요. CRM 트랙은 특히 주의가 필요합니다.

**계정 연결(3교시)은 선택입니다.**
현장에서 하는 연결은 발급이 빠른 2종뿐입니다. 메타 광고 또는 네이버 검색광고 중 하나를 고르시면 되고, 아예 안 하셔도 됩니다. 구글 계열(구글애즈·GA4·구글시트)은 발급 절차가 길어 현장에서 하지 않고 자습 가이드로 드립니다. 연결 없이도 4교시 에이전트와 5교시 액션 제안까지 똑같이 완성됩니다.

**에이전트는 제안까지만 합니다.**
광고 예산 변경, 고객 메시지 발송, 콘텐츠 발행, 데이터 삭제는 사람이 승인한 뒤에만 실행합니다. 계정 연결은 조회 전용이 기본입니다.
