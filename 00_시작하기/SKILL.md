# 시작하기 · 오늘 만들 것과 내 트랙 정하기

> 게이트형 가이드다. 한 번에 다 안내하지 말고 STEP마다 답을 받고 다음으로.
> 이 스킬의 목표는 딱 하나. **수강생이 자기 트랙 번호를 확정하는 것.**
> 여기서 고른 번호가 1교시부터 5교시까지 그대로 따라간다.

## 진행 원칙 (클로드가 지킬 것)

1. 한 STEP씩 안내하고 수강생의 응답을 기다린다.
2. 직군을 못 고르겠다는 사람에게는 STEP 3의 질문 3개로 골라 준다.
3. "틀리면 어쩌지"를 걱정하는 사람에게: 트랙은 중간에 바꿔도 된다고 안심시킨다.
4. 어려운 말은 그 자리에서 쉬운 말로 풀어 준다.

---

## STEP 1. 시작 멘트 (스킬 호출 시 가장 먼저 출력)

```
안녕하세요. 오늘 6시간 동안 만들 것을 먼저 말씀드릴게요.

  ① 내 업무 데이터를 보여주는 대시보드 웹 화면   (2교시)
  ② 명령 한 줄로 그걸 다 해주는 에이전트          (4교시)
  ③ 다음에 뭘 해야 할지 알려주는 액션 제안         (5교시)

오늘의 흐름은 이렇습니다.

  1교시  손풀기      데이터를 눈에 보이는 결과물로 바꿔본다
  2교시  대시보드     내 데이터를 웹 화면으로 만든다
  ─ 점심 ─
  3교시  연결        내 계정에서 실데이터를 가져온다
  4교시  에이전트     명령 한 줄로 전 과정을 돌린다
  5교시  액션·전환    다음 행동을 제안받고, 내 파일로 갈아 끼운다

약속 3가지
  · 터미널·코드는 안 씁니다. 전부 채팅창에서 한국어로.
  · 데이터는 전부 준비돼 있습니다. 아무것도 안 가져오셔도 완주합니다.
  · 결과는 매번 조금 다릅니다. 형태가 같으면 성공입니다.

준비되셨으면 "시작" 이라고 입력해 주세요.
```

⏸ "시작" 응답 대기.

## STEP 2. 내 트랙 고르기

```
직군마다 다루는 데이터가 다릅니다. 그래서 트랙을 나눴어요.
만드는 방법은 전부 같고, 안에 들어가는 데이터와 나오는 결과물만 바뀝니다.

  1  퍼포먼스 마케터   광고 성과 분석         광고비 대비 매출을 본다
  2  콘텐츠 마케터     콘텐츠 기획·성과       뭘 써야 할지, 뭐가 먹혔는지 본다
  3  CRM 마케터        고객 세그먼트·재구매   누가 다시 사는지, 누가 떠나는지 본다
  4  브랜드 마케터     경쟁사·고객의 소리     밖에서 무슨 일이 일어나는지 본다
  5  그로스 마케터     트래픽·퍼널            어디서 손님이 새는지 본다
  6  1인 마케터        전부 조금씩            광고·콘텐츠·매출을 한 화면에 본다

번호 하나만 입력해 주세요. (예: 1)
고르기 어려우시면 "모르겠어요" 라고 답해 주세요. 질문 3개로 찾아드립니다.
```

⏸ 번호 또는 "모르겠어요" 응답 대기.

---

## STEP 3. 못 고르는 사람을 위한 질문 3개

"모르겠어요" 라고 답한 경우에만 진행한다. 한 번에 하나씩 묻는다.

**질문 1**
```
이번 주에 가장 시간을 많이 쓴 일은 무엇이었나요?

  A 광고 성과 확인하고 리포트 만들기
  B 콘텐츠 기획하고 글 쓰기
  C 고객 문의 응대하고 고객 데이터 정리
  D 경쟁사·시장 조사
  E 유입·전환 숫자 들여다보기
  F 위에 있는 걸 다 조금씩
```

**질문 2**
```
지금 손에 있는 데이터는 어느 쪽에 가깝나요?

  A 캠페인별 광고비·클릭·매출
  B 콘텐츠별 조회수·반응
  C 고객 명단·주문 내역
  D 리뷰·댓글·경쟁사 자료
  E 방문·이탈·전환 수치
  F 이것저것 조금씩
```

**질문 3**
```
누가 시키지 않아도 매주 챙겨 보는 숫자는 무엇인가요?

  A 광고 수익률(ROAS)      B 조회수·도달
  C 재구매율               D 브랜드 언급·평판
  E 전환율                 F 이번 주 매출
```

**판정** : A가 많으면 1번, B는 2번, C는 3번, D는 4번, E는 5번, F는 6번 트랙.
답이 갈리면 "가장 시간을 많이 쓰는 일" 기준으로 정해 준다.

```
{트랙명} 트랙을 추천드립니다. 이대로 가시겠어요?
바꾸고 싶으시면 다른 번호를 말씀해 주셔도 됩니다.
```

⏸ 확정 응답 대기.

## STEP 4. 내 트랙 확정 안내

고른 번호에 맞는 내용만 출력한다.

### 1번 퍼포먼스 마케터
```
✅ 1번 퍼포먼스 트랙으로 갑니다.

  실습 데이터   데이터/1_퍼포먼스/
                campaigns.csv   캠페인 30개 성과
                daily.csv       최근 30일 일자별
                creatives.csv   소재 20종

  오늘 만들 것   광고 성과 대시보드 → 광고 분석 에이전트
  5교시 액션     끌 캠페인과 예산 늘릴 캠페인 제안
```

### 2번 콘텐츠 마케터
```
✅ 2번 콘텐츠 트랙으로 갑니다.

  실습 데이터   데이터/2_콘텐츠/
                contents.csv          콘텐츠 40건 성과
                keywords.csv          키워드 100개
                competitor-posts.csv  경쟁사 게시물 30건

  오늘 만들 것   콘텐츠 성과 대시보드 → 콘텐츠 캘린더 에이전트
  5교시 액션     성과 낮은 콘텐츠 제목 리라이트 제안
```

### 3번 CRM 마케터
```
✅ 3번 CRM 트랙으로 갑니다.

  실습 데이터   데이터/3_CRM/
                customers.csv    고객 500명 (익명 처리됨)
                orders.csv       주문 내역
                cs-tickets.csv   문의 60건

  오늘 만들 것   고객 세그먼트 대시보드 → 고객 분석 에이전트
  5교시 액션     떠나려는 고객에게 보낼 메시지 초안

  ⚠️ 개인정보 주의
     실습 데이터는 익명 처리돼 있습니다.
     나중에 본인 데이터를 쓰실 때는 이름·연락처·주소·결제정보를 꼭 지우고 쓰세요.
```

### 4번 브랜드 마케터
```
✅ 4번 브랜드 트랙으로 갑니다.

  실습 데이터   데이터/4_브랜드/
                reviews.csv                    리뷰 200건
                competitor-snapshot-전일.csv   경쟁 3사 어제 상태
                competitor-snapshot-금일.csv   경쟁 3사 오늘 상태
                brand-guide.md                 브랜드 톤 기준
                ad-references.csv              광고 소재 30건

  오늘 만들 것   브랜드 모니터링 대시보드 → 브랜드 모니터링 에이전트
  5교시 액션     경쟁사 새 메시지에 대한 대응안 제안

  💡 어제와 오늘 파일이 따로 있습니다. 두 개를 비교하면 "뭐가 바뀌었는지"가 바로 나옵니다.
```

### 5번 그로스 마케터
```
✅ 5번 그로스 트랙으로 갑니다.

  실습 데이터   데이터/5_그로스/
                sessions.csv     채널별 방문 30일
                funnel.csv       단계별 전환
                ab-results.csv   A/B 실험 3건
                landing.csv      랜딩 페이지 15개

  오늘 만들 것   퍼널 대시보드 → 퍼널 진단 에이전트
  5교시 액션     새는 구간에 대한 개선 가설과 실험 설계
```

### 6번 1인 마케터
```
✅ 6번 1인 마케터 트랙으로 갑니다.

  실습 데이터   데이터/6_1인마케터/
                weekly.csv   광고·콘텐츠·매출 4주치
                tasks.md     반복 업무 15개

  오늘 만들 것   주간 통합 대시보드 → 주간 브리핑 에이전트
  5교시 액션     내 업무 중 무엇부터 자동화할지 진단
```

✅ 체크: "폴더에서 본인 트랙 데이터 파일이 보이시나요?"

## STEP 5. 1교시로 넘기기

```
좋습니다. {번호}번 {트랙명} 트랙으로 확정했습니다.

이 번호는 오늘 끝까지 따라갑니다. 앞으로 나오는 실습은
전부 이 트랙의 데이터로 진행됩니다.

  · 트랙은 중간에 바꿔도 됩니다. 다만 결과물을 처음부터 다시 만들게 됩니다.
  · 본인 회사 데이터가 있으시면 5교시에 갈아 끼웁니다. 지금은 실습 데이터로 갑니다.
  · 회사 보안으로 파일을 못 가져오셔도 괜찮습니다. 실습 데이터로 끝까지 완주합니다.

이제 1교시를 시작합니다. "1교시 실습 시작" 이라고 입력해 주세요.
```

## STEP 6. 자동화 매트릭스 안내 (선택, 30초)

트랙은 정했지만 "내 업무 중에 뭘 자동화해야 하지"가 막막한 사람이 반드시 나온다.
그 사람에게는 4교시에 쓸 도구를 미리 알려 준다. 여기서 진단까지 시키지는 않는다. 1교시가 밀린다.

```
💡 참고로 알려 드릴 게 하나 있습니다.

오늘 4교시에 "내 업무 중 뭘 자동화할지" 정하는 시간이 있습니다.
그때 쓸 진단 도구를 미리 보여 드릴게요.

  마케팅 자동화 매트릭스
  https://likelion-marketing-automation-matrix.netlify.app

내 직군을 고르면 예시 업무가 뜨고, 반복 업무를 넣으면
무엇부터 자동화할지와 그걸 스킬로 만들지 에이전트로 만들지까지 알려 줍니다.

지금 하실 필요는 없습니다. 쉬는 시간이나 점심때 눌러 보셔도 되고,
4교시에 다 같이 해도 됩니다. 미리 해두시면 4교시가 훨씬 빨라집니다.
```

✅ 체크: "링크 열리시나요?" (열리는지만 확인하고 넘어간다)
